# Company Site (local)

Локальный стартовый проект лэндинга компании.

## Структура
- `index.html` — основная страница
- `css/styles.css` — стили
- `js/main.js` — мобильное меню и анимация появления блоков
- `assets/` — изображения и медиа

## Запуск
Открой `index.html` в браузере.

## Дальше
После финальной доработки подключим GitHub-репозиторий и начнем работу через него.
